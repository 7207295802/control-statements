# control-statements

Control statements in Java are code features that control the order of code execution based on logic conditions or repetition cycles. 
Without them, Java code executes strictly from top to bottom, line by line.

*If-Else Statements
if:Executes code only if the condition evaluates to true.
else if: Tests a completely new condition if the previous condition was false.
else: Runs code if none of the preceding conditions evaluate to true

*Switch Statement 
Selects one of many execution blocks based on the exact value of a variable.
It is a cleaner alternative to multiple if-else blocks when checking a single variable against constants

 *Looping Statements 
 Loops repeatedly execute a block of code as long as a specified condition remains true.

* For Loop
 Best utilized when you know exactly how many times the block of code needs to repeat.

*While Loop
Repeatedly runs through code as long as the condition remains true.
 It tests the condition before running the internal loop code.

 *Do-While Loop
 Guarantees the code block will execute at least once because it evaluates the conditional rule after running the internal block.
